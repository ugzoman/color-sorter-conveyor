# Color Sorter Conveyor

This project is a simple color-based sorting system built using **Raspberry Pi**, **Python**, **OpenCV**, and a **servo actuator**.  
The camera detects the object color (e.g. red/green) and a servo motor redirects it to the correct path on a small conveyor.
